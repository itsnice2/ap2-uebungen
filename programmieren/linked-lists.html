Linked Lists

Frage an ChatGPT: Erkläre mir linked lists und zeige mir ein Beispiel in C#

Eine Linked List (verkettete Liste) ist eine Datenstruktur, bei der Elemente (sogenannte „Nodes“) linear angeordnet sind, aber nicht im Speicher hintereinander liegen. Stattdessen verweist jedes Element auf das nächste – dadurch kann man Elemente effizient einfügen oder löschen, ohne den gesamten Speicherblock zu verschieben (wie z. B. bei einem Array).

Eine Linked List besteht aus:

  • Node (Knoten): Enthält die Daten und einen Verweis auf den nächsten Knoten.
  • Head: Zeiger auf den ersten Knoten der Liste.

Es gibt verschiedene Arten:

  • Singly Linked List: Jeder Knoten zeigt nur auf den nächsten.
  • Doubly Linked List: Jeder Knoten zeigt auf den nächsten und vorherigen.
  • Circular Linked List: Der letzte Knoten zeigt zurück auf den ersten.

Beispiel in C#

using System;

class Node
{
    public int Data;
    public Node Next;

    public Node(int data)
    {
        Data = data;
        Next = null;
    }
}

class LinkedList
{
    public Node Head;

    public void Add(int data)
    {
        Node newNode = new Node(data);

        if (Head == null)
        {
            Head = newNode;
        }
        else
        {
            Node current = Head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
        }
    }

    public void PrintList()
    {
        Node current = Head;
        while (current != null)
        {
            Console.Write(current.Data + " -> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

class Program
{
    static void Main()
    {
        LinkedList list = new LinkedList();

        list.Add(1);
        list.Add(2);
        list.Add(3);

        
        list.PrintList();  
    }
}
