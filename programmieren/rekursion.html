Rekursion

Frage an ChatGPT: Erkläre mir Rekursion in der Programmierung, mit einem einfachen Beispiel in C#

Rekursion ist ein Konzept in der Programmierung, bei dem eine Funktion sich selbst aufruft, um ein Problem in kleinere Teilprobleme zu zerlegen. Es wird oft verwendet, um Probleme zu lösen, die sich natürlich in kleinere, ähnliche Probleme unterteilen lassen, wie z. B. die Berechnung von Fakultäten oder das Durchlaufen von Datenstrukturen wie Bäumen.

Eine rekursive Funktion muss immer eine Abbruchbedingung (oder Basisfall) haben, damit sie nicht unendlich weiterläuft. Wenn die Abbruchbedingung erfüllt ist, gibt die Funktion einen Wert zurück, der die Rekursion beendet.

Beispiel: Fakultät, das Produkt aller positiven Zahlen von 1 bis n (in unserem Beispiel 1 bis 5)

5! = 5 x 4 x 3 x 2 x 1 = 120

Beispiel in C#

using System;

class Program
{    
    public static int fakultaet(int n)
    {
        if (n == 0)
        {
            return 1;
        }
        else
        {
            return n * fakultaet(n - 1);
        }
    }


    static void Main(string[] args)
    {
        int zahl = 5;
        int ergebnis = fakultaet(zahl);
        Console.WriteLine($"{zahl}! = {ergebnis}");
    }
}
