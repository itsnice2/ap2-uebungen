Abstrakte Klassen

Frage an ChatGPT: Was sind abstrakte Klassen?

Abstrakte Klassen sind ein Konzept der objektorientierten Programmierung (OOP), das in vielen Programmiersprachen, einschließlich C#, Java und anderen, verwendet wird. Sie dienen als Vorlage für andere Klassen und können nicht direkt instanziiert werden. Stattdessen sollen sie als Basis für abgeleitete Klassen dienen, die die abstrakten Methoden implementieren.

Abstrakte Klassen sind eine Möglichkeit, die Struktur und das Verhalten von Klassen zu definieren, ohne sie vollständig zu implementieren. Sie bieten eine Grundlage für andere Klassen und ermöglichen es, einheitliche Schnittstellen und Funktionalitäten sicherzustellen.

Merkmale abstrakter Klassen

Abstrakte Methoden

Eine abstrakte Klasse kann abstrakte Methoden enthalten, die keine Implementierung haben. Diese Methoden müssen in den abgeleiteten Klassen überschrieben und implementiert werden.

Abstrakte Methoden definieren nur den Methoden-Signatur, nicht die konkrete Ausführung der Methode.

Konstruktormethoden

Abstrakte Klassen können Konstruktoren haben, die in den abgeleiteten Klassen verwendet werden. Sie können auch Felder und Eigenschaften definieren.

Nicht instanziierbar

Eine abstrakte Klasse kann nicht direkt instanziiert werden. Du kannst keine Objekte direkt von einer abstrakten Klasse erstellen, sondern nur von einer konkreten (nicht-abstrakten) Klasse, die von der abstrakten Klasse erbt.

Verwendung zur Vererbung

Abstrakte Klassen dienen häufig dazu, ein gemeinsames Grundgerüst für mehrere Klassen bereitzustellen. Sie ermöglichen es, in abgeleiteten Klassen spezifische Implementierungen zu definieren, während sie gleichzeitig eine gemeinsame Schnittstelle bieten.

Beispiel in C#

using System;


public abstract class Fahrzeug
{
    public string Marke { get; set; }

    
    public abstract void Fahren();
}


public class Auto : Fahrzeug
{
    public int AnzahlTueren { get; set; }

    public Auto(string marke, int anzahlTueren)
    {
        Marke = marke;
        AnzahlTueren = anzahlTueren;
    }

    
    public override void Fahren()
    {
        Console.WriteLine($"Das Auto {Marke} mit {AnzahlTueren} Türen fährt.");
    }
}


public class ElektroAuto : Auto
{
    public int Batteriekapazitaet { get; set; }

    public ElektroAuto(string marke, int anzahlTueren, int batteriekapazitaet)
        : base(marke, anzahlTueren)
    {
        Batteriekapazitaet = batteriekapazitaet;
    }

    
    public override void Fahren()
    {
        Console.WriteLine($"Das Elektroauto {Marke} mit {AnzahlTueren} Türen und {Batteriekapazitaet} kWh fährt umweltfreundlich.");
    }
}

class Program
{
    static void Main(string[] args)
    {
        
        Fahrzeug meinAuto = new Auto("BMW", 4);
        Fahrzeug meinElektroAuto = new ElektroAuto("Tesla", 4, 75);

        
        Console.WriteLine("Fahrzeug 1:");
        meinAuto.Fahren(); 

        Console.WriteLine("\nFahrzeug 2:");
        meinElektroAuto.Fahren(); 

        
        Console.ReadKey();
    }
}
