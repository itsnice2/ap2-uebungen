Generische Klassen

Frage an ChatGPT: Was sind generische Klassen?

Generische Klassen sind in objektorientierten Programmiersprachen wie Java oder C# Klassen, die mit Platzhaltern für Datentypen arbeiten. Diese Platzhalter werden als Generika bezeichnet und ermöglichen es, dass eine Klasse mit verschiedenen Datentypen verwendet werden kann, ohne dass für jeden Datentyp eine separate Klasse erstellt werden muss.

Die Hauptvorteile von generischen Klassen sind:

Wiederverwendbarkeit: Sie ermöglichen es, eine Klasse einmal zu definieren und diese dann mit unterschiedlichen Datentypen zu verwenden.

Typensicherheit: Der Compiler kann während der Kompilierung überprüfen, ob die richtigen Datentypen verwendet werden. Dies reduziert Laufzeitfehler.

Programmiersprachen in denen es generische Klassen gibt:

  • Java
  • C#
  • C++
  • Swift
  • Kotlin
  • TypeScrip
  • Go (Golang)
  • Rust
  • Python

Beispiel in Java


class Box<T> {
    private T value;

    public void setValue(T value) {
        this.value = value;
    }

    public T getValue() {
        return value;
    }
}

public class Main {
    public static void main(String[] args) {
        
        Box<Integer> intBox = new Box<>();
        intBox.setValue(10);
        System.out.println(intBox.getValue());  

        
        Box<String> strBox = new Box<>();
        strBox.setValue("Hallo");
        System.out.println(strBox.getValue());  
    }
}

Beispiel in C#

using System;


class Box<T>
{
    private T value;

    public void SetValue(T value)
    {
        this.value = value;
    }

    public T GetValue()
    {
        return value;
    }
}

public class Program
{
    public static void Main(string[] args)
    {
        
        Box<intT> intBox = new Box<intT>();
        intBox.SetValue(10);
        Console.WriteLine(intBox.GetValue());  

        
        Box<stringT> strBox = new Box<stringT>();
        strBox.SetValue("Hallo");
        Console.WriteLine(strBox.GetValue());  
    }
}
