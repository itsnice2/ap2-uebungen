Abstrakte Klassen vs Generische Klassen vs Interface

generische Klassen abstrakte Klassen Interface
Definition Kapseln Operationen, die nicht für einen bestimmten Datentyp spezifisch sind Definieren ein Standardverhalten Konzentrieren sich auf das Verhalten
Methoden Zwingen Unterklassen zur Implementierung bestimmter Methoden Stellen keine konkreten Methoden bereit
Vererbung Eine Klasse kann von höchstens einer abstrakten Klasse abgeleitet werden Eine Klasse kann beliebig viele Interfaces implementieren
Eigenschaften Können Variablen bereitstellen Können nur Konstantendefinitionen enthalten
Instanziierung wie normale Klasse, mit Typparameter in <> direkt nach Klassennamen Können direkt mit dem Schlüsselwort „new“ instanziiert werden Können nicht direkt instanziiert werden
Verwendung Häufig für Auflistungen wie verknüpfte Listen, Hashtabellen, Stapeln, Warteschlangen Für generische Klassen oder Auflistungsklassen wenn verschiedene Klassen eine gemeinsame Fähigkeit haben sollen, wie zum Beispiel das Drucken oder Vergleichen von Objekten
weiteres Kapseln Operationen, die nicht spezifisch für einen bestimmten Datentyp sind Können als eine besondere Form einer Klasse angesehen werden

Generische Klasse in C#

using System;


public class SpiegelNachrichten
{
    public void NeueNachricht(T empfaenger, string nachricht)
    {
        Console.WriteLine($"Neue Nachricht für {empfaenger.GetType().Name}: {nachricht}");
    }
}



public class User
{
    // ...
}



public class LokalZeitung
{
    // ...
}







class Program
{
    static void Main(string[] args)
    {        
        SpiegelNachrichten userNachrichten = new SpiegelNachrichten();
        User user = new User();
        userNachrichten.NeueNachricht(user, "Willkommen im System!");
        
        SpiegelNachrichten lokalZeitungNachrichten = new SpiegelNachrichten();
        LokalZeitung lokalZeitung = new LokalZeitung();
        lokalZeitungNachrichten.NeueNachricht(lokalZeitung, "Breaking News: Neues Ereignis in der Stadt!");
    }
}

Abstrakte Klasse in C#

using System;


public abstract class SpiegelNachrichten
{
    public abstract void neueNachricht(string Nachricht);
}



public class User : SpiegelNachrichten
{
    public override void neueNachricht(string Nachricht)
    {
        Console.WriteLine("Neue Nachricht für User: " + Nachricht);
    }
}



public class LokalZeitung : SpiegelNachrichten
{
    public override void neueNachricht(string Nachricht)
    {
        Console.WriteLine("Neue Nachricht für LokalZeitung: " + Nachricht);
    }
}




class Program
{
    static void Main(string[] args)
    {
        SpiegelNachrichten user = new User();
        user.neueNachricht("Willkommen im System!");
        
        SpiegelNachrichten lokalZeitung = new LokalZeitung();
        lokalZeitung.neueNachricht("Breaking News: Neues Ereignis in der Stadt!");
    }
}
