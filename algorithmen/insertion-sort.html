Insertion Sort

Der Insertion Sort ist ein einfacher Sortieralgorithmus, der wie das Sortieren von Spielkarten in der Hand funktioniert: Man nimmt ein Element nach dem anderen und fügt es an der richtigen Stelle im bereits sortierten Teil der Liste ein.

Funktionsweise

  • Man geht von links nach rechts durch das Array.
  • Das erste Element gilt als „sortiert“.
  • Für jedes weitere Element:
    • Man vergleicht es mit den vorherigen (sortierten) Elementen.
    • Man schiebt die größeren Elemente eins nach rechts.
    • Dann setzt man das aktuelle Element an die richtige Stelle.

Selection Sort in JavaScript

function insertionSort(zahlen) {
    for (let i = 1; i < zahlen.length; i++) {
        let current = zahlen[i];
        let j = i - 1;

        
        while (j >= 0 && zahlen[j] > current) {
            zahlen[j + 1] = zahlen[j];
            j--;
        }

        
        zahlen[j + 1] = current;
    }

    return zahlen;
