Selection Sort

Selection Sort ist ein einfacher Sortieralgorithmus, der in-place arbeitet und eine Liste schrittweise sortiert, indem er wiederholt das kleinste (oder größte) Element aus dem unsortierten Teil auswählt und es an den Anfang (oder das Ende) des sortierten Teils verschiebt.

Funktionsweise

  • Durchlaufe das Array von vorne bis hinten.
  • Suche das kleinste Element im unsortierten Teil des Arrays.
  • Tausche dieses kleinste Element mit dem ersten Element des unsortierten Teils.
  • Verschiebe den "unsortierten Teil" eins nach rechts.
  • Wiederhole die Schritte, bis das ganze Array sortiert ist.

Selection Sort in JavaScript

function selectionSort(zahlen) {
    const n = zahlen.length;

    for (let i = 0; i < n - 1; i++) {
        
        let minIndex = i;

        
        for (let j = i + 1; j < n; j++) {
        if (zahlen[j] < zahlen[minIndex]) {
            minIndex = j;
        }
        }

        
        if (minIndex !== i) {
        let temp = zahlen[i];
        zahlen[i] = zahlen[minIndex];
        zahlen[minIndex] = temp;
        }
    }

    return zahlen;
}
