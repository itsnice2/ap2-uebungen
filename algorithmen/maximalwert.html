Maximalwert als Pseudocode

Erstelle eine Funktion namens maximalwerte, die ein Array namens werte als Argument entgegennimmt

    Erstelle eine Integer-Variable namens max_wert,  mit dem ersten Wert aus dem Array werte als Inhalt
    Erstelle eine Integer-Variable namens anzahl, mit der Anzahl der Werte aus dem Array werte als Inhalt

    Beginne Schleife: So oft durchlaufen, wie die Anzahl der Werte im Array werte

        Beginne Wenn: Wenn der aktuelle Wert größer ist als der Wert in max_wert

            Gib der Variable max_wert den Wert des aktuellen Wertes

        Ende Wenn

    Ende Schleife

    Gib den Wert der Variablen max_wert zurück.

Ende Funktion

Maximalwert in JavaScript

function maximalwert(werte){
    var max_wert = werte[0];
    var anzahl = werte.length;

    for( var i = 0; i < anzahl - 1; i++){
        if(werte[i] > max_wert){
            max_wert = werte[i];
        }
    }

    return max_wert;
};
