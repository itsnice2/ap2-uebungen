Bubble-Sort

Der Bubble-Sort-Alorithmus vergleicht eine Liste (bzw. Array) von Werten nach und nach mit dem jeweils nachfolgenden Wert. Wenn der Wert größer ist als der nachfolgende Wert, werden die Werte getauscht.

        Beginne Schleife 2: Erstelle eine Variable namens j, und erhöhe sie solange um 1, wie j kleiner ist als die Variable laenge minus Variable i minus 1

Bubble-Sort in JavaScript

function bubbleSort(werte){
    var laenge = werte.length;

    for(var i = 0; i < laenge; i++){

        for(var j = 0; j < laenge - i - 1; j++){

            if(werte[j] > werte[j + 1]){
                var tausch   = werte[j];
                werte[j]     = werte[j + 1];
                werte[j + 1] = tausch;
            }
        }
    }
    return werte;
};

Bubble-Sort in PHP

function bubbleSort(array $werte): array{
    $laenge = count($werte);

    for( $i = 0; $i < $laenge - 1; $i++ ){

        for( $j = 0; $j < $laenge - $i - 1; $j++ ){
            
            if( $werte[$j] > $werte[$j + 1] ){

                $tausch         = $werte[$j];
                $werte[$j]      = $werte[$j + 1];
                $werte[$j + 1]  = $tausch;
            }
        }
    }
    
    return $werte;
}
