Minimalwert als Pseudocode

Erstelle eine Funktion namens minimalwerte, die ein Array namens werte als Argument entgegennimmt

    Erstelle eine Integer-Variable namens min_wert, mit dem ersten Wert aus dem Array werte als Inhalt
    Erstelle eine Integer-Variable namens anzahl, mit der Anzahl der Werte aus dem Array werte als Inhalt

    Beginne Schleife: So oft durchlaufen, wie die Anzahl der Werte im Array werte

        Beginne Wenn: Wenn der aktuelle Wert kleiner ist als der Wert in min_wert

            Gib der Variable min_wert den Wert des aktuellen Wertes

        Ende Wenn

Minimalwert in JavaScript

function minimalwert(werte){
    var min_wert = werte[0];
    var anzahl = werte.length;

    for( var i = 0; i < anzahl - 1; i++){
        if(werte[i] < min_wert){
            min_wert = werte[i];
        }
    }

    return min_wert;
};
